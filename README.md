# CS353 Project

## Server launch

Create a MySQL database with following variables

    PORT=5000
    USER=tutorial
    PASSWORD=test123
    DATABASE=web_app
    DB_PORT=3306
    HOST=localhost

Run the following codes from the terminal:

    npm install
    nodemon index.js

## Client launch

Just open index.html after you have launched the server.
